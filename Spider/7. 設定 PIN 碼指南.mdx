# 設定 PIN 碼指南

![設定 PIN 碼指南1](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FWk98YWM2MHtAzSHwvK6L%2Fblobs%2FjN989mP4Wt62go4kUnjo%2F8.png&width=768&dpr=4&quality=100&sign=49540e43&sv=2)

1.  🏦 資產卡界面:
    進入資產卡界面。
2.  🔑 重設密碼:
    點擊重設密碼進入界面。
3.  🔢 輸入資產卡密碼:
    輸入 6 位數字的資產卡密碼。

![設定 PIN 碼指南2](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FWk98YWM2MHtAzSHwvK6L%2Fblobs%2Flhxm2WCBjyP3EkkH7xx4%2F9.png&width=768&dpr=4&quality=100&sign=a19000aa&sv=2)

4.  ✅ 確認重設:
    點擊重設密碼。
    點擊 OK，確認完成更改。
5.  🎉 完成:
    完成後，會顯示卡片 PIN 碼更改成功提示。
