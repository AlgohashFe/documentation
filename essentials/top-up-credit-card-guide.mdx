# 充值額度至信用卡指南

![充值額度至信用卡指南1](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2F576526375-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FWk98YWM2MHtAzSHwvK6L%252Fuploads%252FBwAXAPAGTy0AMeae5vhE%252F67.png%3Falt%3Dmedia%26token%3D223dbf98-237e-4882-845f-9b7d2b06f48c&width=768&dpr=4&quality=100&sign=ea987602&sv=2)

1.  💼 首先按"資產卡"。

![充值額度至信用卡指南2](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2F576526375-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FWk98YWM2MHtAzSHwvK6L%252Fuploads%252FleHfsYSRKPJqQyUZLdME%252F68.png%3Falt%3Dmedia%26token%3D0d7cdf75-c12e-4aa8-b09d-a987d8dc3464&width=768&dpr=4&quality=100&sign=41753529&sv=2)

2.  💰 進入資產卡畫面後,按"新增信貸額度"。

![充值額度至信用卡指南3](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2F576526375-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FWk98YWM2MHtAzSHwvK6L%252Fuploads%252F2kfLtUwIfJzZBdeDiFqW%252F69.png%3Falt%3Dmedia%26token%3D398f2be7-31a7-4330-aeda-45b53ca0e3e4&width=768&dpr=4&quality=100&sign=627d9527&sv=2)

3.  💵在支付USDT的空間中輸入增值額度。

![充值額度至信用卡指南4](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2F576526375-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FWk98YWM2MHtAzSHwvK6L%252Fuploads%252FgE32i4oQibfel83eFdFS%252F70.png%3Falt%3Dmedia%26token%3D6684f669-388e-4b94-8da4-77fbcc5e904b&width=768&dpr=4&quality=100&sign=bad835cf&sv=2)

4.  💵 輸入增值額度後,按"報價"。

![充值額度至信用卡指南5](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2F576526375-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FWk98YWM2MHtAzSHwvK6L%252Fuploads%252F2PkBFwSnwr1ZrZg6mK9P%252F71.png%3Falt%3Dmedia%26token%3Dc8ebb2f1-4044-4ea4-82f2-a8fd6de2d350&width=768&dpr=4&quality=100&sign=c99f874d&sv=2)

5.  📊 查看報價是否合宜,按"接受報價並充值卡"。

![充值額度至信用卡指南6](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2F576526375-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FWk98YWM2MHtAzSHwvK6L%252Fuploads%252FDwFCyeTyhAGD8SSnqLnt%252F72.png%3Falt%3Dmedia%26token%3D303a4842-2c75-43d3-9a9b-b59571cb7bd1&width=768&dpr=4&quality=100&sign=660c08fd&sv=2)

6.  🎉 成功報行報價後會顯示"報價已成功執行"提示。
