---
title: 'Setting Up 2FA Security Code'
description: 'How to set up Two-Factor Authentication (2FA) in the SpiderCard app.'
---

<Info>
Two-Factor Authentication (2FA) is a crucial security measure for protecting your SpiderCard account. With 2FA enabled, you'll need to enter a dynamic verification code from an authenticator app (like Google Authenticator) in addition to your password to log in or perform sensitive operations, significantly enhancing your account security. We strongly recommend all users enable 2FA.
</Info>

![Setting Up 2FA 3](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2F576526375-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FWk98YWM2MHtAzSHwvK6L%252Fuploads%252Fn68i8Mi9c2ljQ1sj1Ja7%252F377.png%3Falt%3Dmedia%26token%3D7630db9b-9a5c-45c6-9646-469319a114c0&width=768&dpr=4&quality=100&sign=bddfb6db&sv=2)
![Setting Up 2FA 4](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2F576526375-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252FWk98YWM2MHtAzSHwvK6L%252Fuploads%252Farn5YbUoakZ4dNgYtavT%252F378.png%3Falt%3Dmedia%26token%3Da3e575e3-d857-4921-9068-ee951775e11c&width=768&dpr=4&quality=100&sign=6032c144&sv=2)

3. üìã The system will display a **key** for setting up 2FA (usually provided as both QR code and text). Long press the text key and tap **"Copy"**.

<Warning>
**EXTREMELY IMPORTANT: Make sure to backup this key in a secure location!**
* This key is your only way to recover 2FA access if you lose or replace your phone.
* Write down the key by hand and store it in a safe place, or save it in a secure password manager.
* Do NOT store the key as a screenshot or plain text file on any connected device.
* Without a backup key, you might permanently lose access to your SpiderCard account if you can't access your authenticator app!
</Warning>

6. üìù Return to Spider app:
   Return to the Spider app, use **"Paste" or "Auto-fill"** to enter the verification key, then enter the 6-digit code generated by Google Authenticator.
7. üü¢ **Enable 2FA**: Click the **"Activate 2FA"** button. If the verification code is correct, two-factor authentication will be successfully enabled. After successful activation, you should see a green "2FA enabled" indicator or similar notification at the bottom of the interface.

Congratulations! You have successfully added an extra layer of security to your SpiderCard account. Remember to keep your backup key safe.
