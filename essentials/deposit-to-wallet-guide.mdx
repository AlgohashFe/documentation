# 充值到錢包指南

![充值到錢包指南1](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FWk98YWM2MHtAzSHwvK6L%2Fblobs%2FbIaRzPGERHlKnBLIVzHC%2F2.png&width=768&dpr=4&quality=100&sign=4d73eebb&sv=2)

1.  💼 造訪錢包主頁，點擊**"存入抵押品"**按鈕進入存款界面。
2.  🔄 點擊**"進行存款"**繼續操作。

![充值到錢包指南2](https://milways.gitbook.io/~gitbook/image?url=https%3A%2F%2Fcontent.gitbook.com%2Fcontent%2FWk98YWM2MHtAzSHwvK6L%2Fblobs%2Fum4rJVKVv0uWvP7tNHpc%2F3.png&width=768&dpr=4&quality=100&sign=ac44496c&sv=2)

3.  🌐 選擇網絡 : 在二維碼下方，您可以選擇使用 Tron（TRX）或 Ethereum（ETH）或 Solana (SOL) 等網絡。
4.  🔢 獲取存款地址 : 如果選了TRX，系統將顯示一個二維碼和 TRX 網絡地址，點擊地址下方的**"複製"**按鈕。
